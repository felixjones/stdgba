# Audio Overview

stdgba audio currently focuses on PSG composition via `gba/music`: compile-time Strudel-style patterns that play on the GBA's four hardware channels.

- Start here: [Music Composition](./music.md)
- Learn sample and custom-wave workflows in [Embedded WAV Samples](./wav-embed.md)
- Channel guides:
  - [Square 1 (CH1)](./channel-sq1.md)
  - [Square 2 (CH2)](./channel-sq2.md)
  - [Wave (CH3)](./channel-wav.md)
  - [Noise (CH4)](./channel-noise.md)

For syntax and semantics, see Strudel docs linked from `music.md`.
